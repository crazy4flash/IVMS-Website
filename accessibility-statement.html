Accessibility Statement

Our commitment to making our website accessible to all users

Last Updated: December 2024

Compliance Status

IVMS is committed to ensuring digital accessibility for people with disabilities. We are continually improving the user experience for everyone and applying the relevant accessibility standards.

Conformance status: The Web Content Accessibility Guidelines (WCAG) defines requirements for designers and developers to improve accessibility for people with disabilities. It defines three levels of conformance: Level A, Level AA, and Level AAA. IVMS is partially conformant with WCAG 2.1 level AA. Partially conformant means that some parts of the content do not fully conform to the accessibility standard.

Accessibility Features

What We've Implemented

  • Semantic HTML structure with proper heading hierarchy
  • Alternative text for all images and non-text content
  • Keyboard navigation support throughout the website
  • High contrast color schemes and sufficient color contrast ratios
  • Resizable text support (up to 200% without loss of functionality)
  • Focus indicators for all interactive elements
  • Skip navigation links for screen reader users
  • Form labels and error messages for screen readers
  • Responsive design that works on all device sizes
  • Support for screen readers and assistive technologies
  • Reduced motion support for users with vestibular disorders
  • High contrast mode support

Known Accessibility Issues

We are aware of the following accessibility issues and are working to resolve them:

  • Some video content may not have complete captions or audio descriptions
  • Complex data visualizations may not be fully accessible to screen readers
  • Some third-party integrations may not meet our accessibility standards

We are actively working to address these issues and improve the overall accessibility of our website.

Assistive Technologies

Our website has been tested with the following assistive technologies:

  • Screen readers: NVDA, JAWS, VoiceOver
  • Screen magnification software
  • Voice recognition software
  • Keyboard-only navigation
  • High contrast mode

Browser Support

Our website is designed to work with the following browsers and their accessibility features:

  • Chrome (latest version)
  • Firefox (latest version)
  • Safari (latest version)
  • Edge (latest version)

Accessibility Tools

We recommend the following tools to enhance your browsing experience:

  • Screen Readers: NVDA (free), JAWS, VoiceOver (built into macOS)
  • Magnification: Built-in browser zoom, Windows Magnifier, macOS Zoom
  • High Contrast: Windows High Contrast mode, macOS Dark Mode
  • Keyboard Navigation: Use Tab, Shift+Tab, Enter, and arrow keys to navigate

Technical Specifications

Accessibility of our website relies on the following technologies to work with the particular combination of web browser and any assistive technologies or plugins installed on your computer:

  • HTML5
  • CSS3
